# Expression Tree 表达式树

```
/* 表达式树 */
/* 叶结点是运算数，非叶结点是运算符 */

+
|\
+     *
|\    |\
a *   + g
  |\  |\
  b c * f
      |\
      d e


/* 不同的遍历，得到不同的访问结果 */

// 1. 中缀表达式(LVR)：(a + (b * c)) + (((d * e) + f) * g)
// 2. 前缀表达式(VLR)：++a*bc*+*defg
// 3. 后缀表达式(LRV)：abc*+de*f+g*+
```
